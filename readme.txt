Onward journeys plugin.

== Description ==

Allows editors to define onward journeys within their articles.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use this shortcode in your article.

`
[onward-journeys]
[This is the link text|http://this.is.the.link.com]
[This is another link - you can have as many as you like|http://this.is.another.link.com]
[/onward-journeys]
`
